列表

有序列表

数字 + 英文句点（.）
比如：

  1. 你好1
  2. 你好2
  3. 你好3

无序列表

使用（-,*,+）均可 如：

  • 无序1
  • 无序2
  • 无序3

还可以嵌套，缩进即可，如：

  • 一级A
  • 一级B
    • 二级BA
    • 二级BB
  • 一级C

代码块

四个空格或者一个tab
如果在列表中，使用8个空格或两个tab

tips：注意代码前面一定留一个空行

比如正常code：

private static final String STR_EMPTY = "";

图片

要添加图片，首先请添加感叹号（!），然后紧跟着是方括号，方括号中可添加替代文本（alt text，即图片显示失败后显示此文本），最后跟着圆括号，圆括号中添加图片资源的路径或 URL。你可以选择在圆括号中的 URL 之后添加标题（即 title 属性）。

单词/代码

使用反引号

分割线

使用三个或多个（***,---,___） 如：

—

链接

方括号写链接展示文本，后面圆括号写url 如： 请打开我的首页

快速转链接

使用<>包括链接
比如：
https://xiaohanqing.github.io/ fake@example.com
